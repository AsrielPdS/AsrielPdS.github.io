CRUD, não usado nenhuma biblioteca externas (isto é, todos os comportamentos e estilos foram implementados por mim).

  • Visualização de dados, permite tambem cabeçararios e rodapes diferentes para paginas impar e par, e para primeira pagina
  • Criação, renderiza o documento diferente baseado em uma ou mais fontes de dados.
  • Edição, diferentes partes do documento podem utilizar expressões baseada em planilhas electronicas(como o excel).
  • Pesquisa, os dados de uma tabela ou lista podem ser preenchidos apartir de uma fonte de dados (tal como as listas de itens em uma factura).
  • Filtros, tamanho e orientação podem ser costumizados tambem de suporte para impressores de só uma paginas(impressoras de talão), é possivel mostrar ou esconder determinados itens baseado no formato do papel.
  • Reordenar, permite alterar a moeda que que se deseja usar no documento e pode se definir deversas versões do documento para multiplas linguas.
  • Impressão, permiti criar diferentes, isto é util quando se deseja fazer um mesmo modelo de documento para diversos clientes (empresas) diferentes.
