Biblioteca para renderização de documentos paginados para fins de impressão, com as seguintes ferramentas.

  • Cabeçario e rodape, permite tambem cabeçararios e rodapes diferentes para paginas impar e par, e para primeira pagina
  • data binding, renderiza o documento diferente baseado em uma ou mais fontes de dados.
  • formulas, diferentes partes do documento podem utilizar expressões baseada em planilhas electronicas(como o excel).
  • tabelas e listas dinamicas, os dados de uma tabela ou lista podem ser preenchidos apartir de uma fonte de dados (tal como as listas de itens em uma factura).
  • diferentes tipos de formato de papel, tamanho e orientação podem ser costumizados tambem de suporte para impressores de só uma paginas(impressoras de talão), é possivel mostrar ou esconder determinados itens baseado no formato do papel.
  • moeda e lingua, permite alterar a moeda que que se deseja usar no documento e pode se definir deversas versões do documento para multiplas linguas.
  • temas, permiti criar diferentes, isto é util quando se deseja fazer um mesmo modelo de documento para diversos clientes (empresas) diferentes.

Obs: Esta ferramenta não usa bibliotecas externas (Todos os comportamentos foram implementados por mim).
